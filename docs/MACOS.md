# macOS 기본값

`chezmoi/run_once_set-macos.sh.tmpl`에서 기본값을 설정합니다.
개발 편의성과 예측 가능한 동작을 우선한 설정입니다.

## 설계 기준

- **일관성**: macOS와 리눅스 간 동작 차이로 생기는 혼란 최소화
- **가시성**: 경로/확장자/숨김 파일을 명시적으로 보이게
- **개발 친화**: 코드 작성에 불리한 스마트 기능 비활성화
- **환경 분리**: headless(서버)와 desktop(맥북/데스크탑) 옵션 분리

## 적용되는 항목과 이유

### Finder/파일

- 확장자 표시: 동일 파일명 혼동 방지
- 경로/상태 막대 표시: 파일 위치와 상태를 빠르게 확인
- 숨김 파일 표시: dotfiles 편집을 쉽게
- 폴더 우선 정렬: 대규모 디렉터리에서 탐색 효율 향상
- 검색 범위 기본값을 현재 폴더로: 프로젝트 내 검색 실수 감소
- 타이틀에 전체 경로 표시: 경로 착각 방지
- 기본 보기(리스트): 개발자 입장에서 정보 밀도가 높음
- 퀵룩 텍스트 선택: 빠른 복사/조회

### 입력/키보드

- 스마트 입력 비활성화: 코드 작성 시 따옴표/대시 변형 방지
- 키 반복 속도 개선: 편집/이동 속도 향상
- 길게 누르기 비활성화: 키 반복을 우선

### 저장/인쇄

- 저장/인쇄 패널 기본 확장: 파일 경로/옵션 접근성 향상
- iCloud 저장 기본값 해제: 로컬 저장을 기본으로

### 스크린샷

- 저장 위치 고정: 파일 관리가 쉬워짐
- PNG, 그림자 비활성화: 문서/이슈 첨부 시 깔끔한 결과

### Dock/Spaces

- 최근 앱 숨김: Dock의 시각적 노이즈 감소
- 앱별 최소화: 창 관리 일관성
- 공간 자동 재배치 비활성화: 작업 공간의 예측 가능성 확보

### 파일 시스템

- 네트워크/USB의 .DS_Store 생성 방지: 공유 디렉터리 오염 방지
- Library 폴더 표시: 설정 접근성을 높임

### headless(서버)

- 슬립/디스플레이 슬립 비활성화: 원격 접속 안정성
- 자동 재시작: 무인 운영 신뢰성
- Dock 자동 숨김: VNC 화면 확보

### desktop(맥북/데스크탑)

- 기본 절전: 배터리 사용 고려
- 탭 투 클릭 활성화: 트랙패드 사용 편의

## 변경/되돌리기

1) `run_once_set-macos.sh.tmpl`에서 원하는 항목 수정
2) `chezmoi apply`로 다시 적용

스크립트는 Finder/Dock/SystemUIServer를 재시작합니다.
특정 항목을 되돌리려면 `defaults delete`로 개별 키를 삭제하세요.
