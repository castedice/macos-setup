# Starship

`chezmoi/dot_config/starship.toml.tmpl`에서 프롬프트를 설정합니다.

## 기본 정책

- headless(헤드리스): 최소 정보(디렉터리/깃 상태)
- desktop(데스크탑): 전체 정보(사용자/호스트/디렉터리/깃/시간)

헤드리스는 SSH 접속 위주라 짧고 빠른 프롬프트를 우선합니다.

## 변경 방법

- `format` 문자열을 원하는 형태로 수정
- 모듈별 설정을 추가/제거

추가로 활성화된 모듈:

- jobs: 백그라운드 작업 수 표시
- status: 실패한 명령의 상태 코드 표시
- directory: 읽기 전용 디렉터리 표시

적용 후 새 터미널에서 확인됩니다.
