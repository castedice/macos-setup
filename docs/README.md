# 문서 인덱스

이 폴더에는 각 요소별 설정과 커스터마이징 방법, 선택 이유/대안/트레이드오프를
정리했습니다. 기본 설정을 그대로 쓰더라도 “왜 이렇게 했는지”를 이해할 수 있도록
가이드 성격으로 작성했습니다.

- `BOOTSTRAP.md`: 부트스트랩 흐름과 변경 포인트
- `CHEZMOI.md`: chezmoi 구조/프롬프트/롤백
- `BREW.md`: Homebrew 패키지 구성과 추가 방법
- `RUNTIMES.md`: 언어 런타임 선택 가이드
- `DIRENV.md`: direnv 사용법
- `ZSH.md`: zsh + sheldon 설정
- `STARSHIP.md`: 프롬프트 스타일
- `GIT.md`: Git 기본값 및 SSH 서명
- `NVIM.md`: Neovim 기본 설정/플러그인
- `MACOS.md`: macOS 기본값 적용
